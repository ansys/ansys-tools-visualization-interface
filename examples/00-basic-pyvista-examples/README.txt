Basic usage examples
====================

These examples show how to use the general plotter included in the Visualization Interface Tool.

New: Customization APIs
------------------------

See ``customization_api.py`` for examples of the new backend-agnostic customization APIs:

- ``add_points()`` - Add point markers
- ``add_lines()`` - Add line segments
- ``add_planes()`` - Add reference planes
- ``add_text()`` - Add text labels

These APIs work consistently across different backends.
