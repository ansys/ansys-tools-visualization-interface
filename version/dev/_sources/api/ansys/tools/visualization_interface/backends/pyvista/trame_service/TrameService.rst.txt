.. py:class:: ansys.tools.visualization_interface.backends.pyvista.trame_service.TrameService(websocket_host: str = 'localhost', websocket_port: int = 8765)

   
   Trame service class.

   Initializes a trame service where you can send meshes to plot in a trame webview plotter.

   :Parameters:

       **websocket_host** : :class:`python:str`, :obj:`optional`
           Host where the webserver will listen for new plotters and meshes, by default "localhost".

       **websocket_port** : :class:`python:int`, :obj:`optional`
           Port where the webserver will listen for new plotters and meshes, by default 8765.
